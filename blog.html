Scope

Scope of "Var"

Var is a global Scope . We can access this from anywhere.
But in a function we can't access it! We can reassign this and redeclare this.

Scope of "let"

let is a block Scope .We can access this from only it's own scope.
We can reassign this but can't redeclare this.

Scope of "Const"

Const is a block Scope .We can access this from only it's own scope.
We can't reassign & redeclare this.

Use case Null & Undefined

When an array element or object property value is not assigned and if we want to access them then we will get the value undefined. There are other undefined values in JavaScript, such as - if no parameter is passed to a function, there is no return value but if you want to access it, if you want to access a variable without defining its value, undefined comes. If an array or object has a missing value then the null value is used.

REST API

REST is Representational State Transfer and Its an architectural style which is used for websites and API is Apploication Programing Interface. REST API is a way of accessing web services in a simple and flexible way without having any processing. REST API uses HTTP(Hyper Text Transfer Protocol) request. In HTTP there are five methods that are commonly used in a REST-based Architecture i.e., POST, GET, PUT, PATCH, and DELETE. These correspond to create, read, update, and delete (or CRUD) operations respectively.
